# Review Prompts

## Security review

```text
Review the purchase flow for ways a malicious player could get an item without paying. Check duplicate tx, wrong memo, wrong amount, wrong mint, wrong destination, expired order, and replay attacks. Provide exact code changes.
```

## Game design review

```text
Review whether the 5-minute demo is fun for a non-crypto player. Suggest only changes that fit a 7-day browser prototype.
```

## Pump.fun messaging review

```text
Review the token launch copy for legal/reputational risk. Remove ROI language, yield language, price targets, passive income framing, revenue share claims, and pay-to-win utility.
```
