# 12 — Asset and Audio Plan

## Prototype rule

Use generated shapes or CC0 placeholder assets until the game loop is fun. Do not spend heavily on art before player retention is tested.

## Visual requirements

```text
Base resolution: 320x288
Tile size: 16x16
Hero sprite target: 16x24
NPC target: 16x24
Enemy sprite target: 32x32 or 48x32
UI: thick pixel borders
Palette: limited GameBoy-inspired colors
```

## Minimum prototype assets

```text
hero idle/walk placeholder
NPC placeholder
shopkeeper placeholder
dungeon gate
3 enemy placeholders
1 boss placeholder
UI frame
Aether crystal icon
```
