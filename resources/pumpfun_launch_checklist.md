# Pump.fun Launch Checklist

## Before creating token

- [ ] Demo link live
- [ ] Gameplay clip ready
- [ ] Token name/ticker final
- [ ] Image readable at small size
- [ ] Description has no ROI/yield/profit promises
- [ ] Official links ready
- [ ] Mint verification post draft ready
- [ ] Community channel ready
- [ ] Team/creator wallet disclosure policy ready
- [ ] Treasury/creator fee usage policy ready
- [ ] No paid power utility promised

## Immediately after token creation

- [ ] Copy mint address
- [ ] Pin official mint address
- [ ] Post launch thread
- [ ] Post risk/utility statement
- [ ] Warn about fake mints
- [ ] Add token mint to website/demo config only after verified

## First week

- [ ] Ship demo patch
- [ ] Publish devlog
- [ ] Show cosmetic concept
- [ ] Track wallet conversion
- [ ] Track community sentiment
- [ ] Avoid price target posts
- [ ] Avoid buyback/revenue share promises
